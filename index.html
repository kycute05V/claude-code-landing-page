Chương Trình 8 Buổi

16 tiết học thực chiến, hoàn thành project có thể deploy

Hướng dẫn cài đặt Claude Code, cấu hình môi trường, làm quen với giao diện và các lệnh cơ bản. Thực hành luồng làm việc đầu tiên.

Các kỹ thuật viết prompt nâng cao: cấu trúc, ngữ cảnh, giới hạn output, và best practices để Claude Code hiểu đúng ý bạn.

Cùng AI phân tích yêu cầu dự án, thiết kế kiến trúc Firestore, Firebase Auth, Firestore Rules, và cấu trúc collection. Thực hành trên ProjectOS.

Thiết lập Firebase project, Firebase Auth (email/password, Google), Firestore database, Firebase Functions, Security Rules. Tạo API endpoints đầy đủ.

Xây dựng giao diện với Tailwind CSS, kết nối Firebase Auth & Firestore, xử lý các nghiệp vụ CRUD, real-time listeners, và form validation.

Sử dụng Claude Code để phân tích bug, trace lỗi qua stack trace, refactor code sạch hơn, và tối ưu hiệu suất truy vấn Firestore.

Tích hợp Firebase Storage cho upload file, Firebase Cloud Messaging cho thông báo, Firebase Analytics, và các tính năng premium khác.

Deploy ứng dụng lên Vercel, cấu hình custom domain, tối ưu SEO, Lighthouse audit, và hoàn thiện portfolio cá nhân với project đã deploy.
